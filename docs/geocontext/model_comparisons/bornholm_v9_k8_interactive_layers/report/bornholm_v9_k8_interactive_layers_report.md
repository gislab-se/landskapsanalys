# Bornholm v9: K=8 lagerkarta

## Syfte

v9 ar en pedagogisk publiceringsversion for att visa K=8-losningen fran v3/v1-sparet. Kartan har ett lager for alla kluster, valbara individuella kluster, faktorlager for F1-F5 och en opacitetskontroll for det aktiva lagret.

## Versioner v1-v9

| Version | Kort beskrivning |
|---|---|
| v1 | Forsta hogre-H3/local-context-sparet. K=7 blev en stark kandidat och gav tydlig landskapsstruktur. |
| v2 | Skalad k-variant. Gav ringartefakter och anvands inte som huvudtolkning. |
| v3 | K-genomgang pa v1/v3-underlaget. K=8 ar pedagogiskt stark for sprickdalarna. |
| v4 | Temabalanserade indata. F1 visar sprickdalar/brant relief tydligt som faktor. |
| v5 | Temabalanserad K-genomgang. K=9 visar sprickdalarna fint men behovde tydligare legendnamn. |
| v6 | Mean-only K=8 som diagnostik. Visade att standardavvikelserna bar viktig lokal heterogenitet. |
| v7 | Pedagogisk demo med K=5 till K=10 och sprickdalsreferenser fran v4/v5. |
| v8 | Reserverad mellanversion utan separat publicerat run-id. Anvandes inte som egen modellkandidat. |
| v9 | Denna publicerade lagerdemo: fast K=8, alla kluster, individuella klustertoggles, faktorlager och opacitet. |

## Faktorer

- F1: Sprickdal och brant relief.
- F2: Flygsand och sandpraglad kust.
- F3: Skog och skyddad natur.
- F4: Tatort och byggd struktur.
- F5: Laglant oppet land.
