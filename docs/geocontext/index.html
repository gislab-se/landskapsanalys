Landskapsanalys Bornholm

Här samlas den publicerade landskapsanalysen för Bornholm tillsammans med kartor, vindpotential, den äldre scenarioappen och potentialapp v1 för sol och vind. Sidan är byggd som en enkel ingång till att först förstå modellen och sedan använda den i fortsatt analys.

Börja med rapporten om du vill få grepp om modellen. Gå sedan vidare till vindpotential, den äldre scenarioappen eller potentialapp v1 om du vill använda den.
